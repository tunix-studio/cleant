## [0.1.0] — unreleased

First public release.

### Added

- **Overview / Smart Scan** with an ocean‑themed disk‑health score and factor breakdown.
- **Clean** engine: caches, logs, developer junk, browser data, Trash — move‑to‑Trash
  by default, confirmation, path guards; expandable rows with “open location”.
- **Storage** breakdown.
- **Large & old files** finder (size/age sort).
- **Duplicate** finder (content hash; keep one per set).
- **Developer** cache cleanup (Xcode, simulators, npm/Gradle/CocoaPods/Homebrew, …).
- **Leftovers** finder (orphaned app files + dead PATH entries).
- **Apps**: Homebrew updates, uninstall with leftovers, per‑app footprint.
- **Performance**: startup/login items + internet speed test.
- **Cleanup reminders** + optional **scheduled auto‑clean**.
- **Cleanup history** and **exclusion rules** (enforced natively).
- Menu‑bar (tray) mode, notifications, onboarding, light/dark themes.
